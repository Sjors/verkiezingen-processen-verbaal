# Amersfoort

Source: https://www.amersfoort.nl/verkiezingsuitslagen (sneltelling PVs)
and https://www.amersfoort.nl/cso-uitslagen-gemeenteraadsverkiezing-2026 (CSO hertellingen)

Amersfoort uses centraal stemopneming (CSO) for the definitive count.
Both sneltelling (sb-*) and CSO (cso-*) processen-verbaal are archived.
