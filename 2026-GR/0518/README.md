# 's-Gravenhage (Den Haag)

## Opmerkingen

Per-stembureau N 10-1 PV PDFs are **not** available for download from this municipality.

Only summary documents are available:
- GSB (Gemeentelijk Stembureau) PV
- CSB (Centraal Stembureau) PV  
- New session PV
- Corrigendum
- Verklaring aansluitvoorschriften
- Verslag controleprotocol

Individual vote data by polling station is viewable online at https://uitslagen.denhaag.nl/ but per-stembureau PDF documents are not published.
