# Arnhem

Source: https://www.arnhem.nl/alle-onderwerpen/verkiezingen/telling-en-processen-verbaal-gemeenteraadsverkiezingen-2026/

Arnhem publishes grouped processen-verbaal PDFs covering stembureau ranges, with separate bundles
for `per stembureau op lijst` and `per stembureau op lijst en kandidaat`, plus the gemeentelijk
stembureau report and appendix.

|------------|-------------|
| 2023-TK    | 23          |
| 2026-GR    | 20 grouped PDFs |
