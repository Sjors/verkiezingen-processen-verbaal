# Wormerland

Wormerland gebruikt centraal tellen: op verkiezingsavond worden per stembureau de partijstemmen geteld (eerste telling). Op donderdag 19 maart worden alle stemmen per kandidaat centraal geteld door het gemeentelijk stembureau.
